SafeWalk Project

What is SAFEwalk?

"Macalester's SAFEwalk is a campus program dedicated to providing the college's students, faculty, and staff a safe campus environment." SAFEwalk is a Macalester College community service which allows all community members to be safely escorted on or off campus.
For more information see SAFEwalk website.

SafeWalk Android Application

SafeWalk App is an android application which provides access to Macalester SAFEwalk services. Through a simple and intuitive interface, the users are able to place pick-up requests which are directly sent to SAFEwalk.

SafeWalk webpage

This module of the SafeWalk project allows the administration of Macalester SAFEwalk services to remotely manage the request placed through the above android application. This module has been created for private use of the SAFEwalk administration.
